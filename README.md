# Streamer.Bot Android App Instructions
1. Download the latest APK from Releases and Install on your tablet
2. In your Streamer.Bot Instance on your computer, naviagte to Servers/Clients > Websocket Server
3. Set the "Address" to 0.0.0.0 and start the server
4. Open the app on your phone and tap connect
5. In the popup box, enter the IP Address of your computer, the port (defaults to 8080) and the password if you set one in Streamer.Bot
6. Tap Connect
7. If all goes well, the icon should turn green and all your actions will auto populate as buttons
